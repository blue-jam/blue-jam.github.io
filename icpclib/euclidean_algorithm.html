ユークリッドの互除法

説明

最大公約数(GCD:Greatest Common Divisor)を求めるためのアルゴリズム． 最大公約数がわかれば最小公倍数(LCM: Least Common Multiple)が求められる．

計算量

O(log max(a, b))

使用方法

0以外の数を与えると，GCD，LCMを計算する． 負の数については動作を確認していない．

ソースコード

ll gcd(ll x, ll y){
    return y == 0 ? x: gcd(y, x % y);
}
ll lcm(ll x, ll y){
    return x / gcd(x, y) * y;
}
