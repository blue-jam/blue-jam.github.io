直線，線分の交差判定，交点計算

説明

直線の交差判定については簡単である．ベクトルa，bで表される直線とベクトルc，dで表される直線が与えられたとき，cross(b - a, d - c)が0であれば，2つの直線は平行であるから，重なっていない限り交差はしない．

使用方法

intersection

直線，線分の交点計算．2直線が平行でないことが前提．

ソースコード

P intersection(const Line &a, const Line &b){
    P va = a[1] - a[0], vb = b[1] - b[0];
    return a[0] + va * cross(vb, b[0] - a[0]) / cross(vb, va);
}
