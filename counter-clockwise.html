点の進行方向の判定

説明

3点a,b,cをa->b->cの順番で進むときに，反時計方向に進む（b-aの左側方向へ進む）か，時計方向に進む（b-aの右側の方向へ進む）か判定する．

使用方法

3点を与える．反時計方向であれば1，時計方向であれば-1，平行であれば0を返す．

ソースコード

int ccw(P a, P b, P c){
    P ab = b - a, ac = c - a;
    if(cross(ab, ac) > eps)     return +1;      //counter clockwise
    if(cross(ab, ac) < -eps)    return -1;      //clockwise
    return 0;                                   //parallel
}
